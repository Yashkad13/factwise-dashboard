# FactWise Employee Dashboard

A clean, professional employee dashboard built with **React** and **AG Grid** (client-side rendering).

## Features
- AG Grid table with sorting, resizing, and pagination
- Search bar — filter by name, position, location
- Department dropdown filter
- Summary stat cards (Total, Active, Avg Salary, Avg Rating)
- Custom cell renderers — status badges, rating colors, skill tags

## How to Run

1. `npm install`
2. `npm start`
3. Open http://localhost:3000
